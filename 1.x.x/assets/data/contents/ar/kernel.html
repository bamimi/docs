# Kernel Structure

# The Auth Directory

The auth directory contains code initialize instance for auth, (here using @knfs-tech/bamimi-auth).

# The Cache Directory

The cache directory contains code initialize instance for cache, have two types: Redis and local.

# The Database Directory

The database directory contains code initialize instance for database.

# The File Directory

The file directory contains code initialize instance for file and function to work with file.

# The Interface Directory

The interface directory contains code initialize and configuration version for Web UI and API.

# The Job Directory

The job directory contains code initialize instance for jobs and schedule.

# The Log Directory

The log directory contains code initialize instance for logging.

# The Mail Directory

The mail directory contains code initialize instance for mail and function to work with mail.

# The Queue Directory

The queue directory contains code initialize instance for queue.

# The Request Directory

The request directory contains code initialize instance for request.

# The Response Directory

The response directory contains code initialize instance for response.

# The Router Directory

The router directory contains code initialize instance for HTTP request to two web endpoint and api endpoint.

# index.js

index.js contains code initialize instance for common HTTP request and two web endpoint and api endpoint.
