Utils

# Introduction

BAMIMI supports you with a number of convenient ready-to-use utilities

# Asset

You can configure paths to assets in multiple places, by passing configs/app.js in, and using $_asset(...) in WebUI

# Paginate

A small utility that calculates the pagination for you

# Url

You can configure it in configs/app.js, and use $_url(...) in webUI, it helps to handle problems when forming absolute urls combined with query params.
